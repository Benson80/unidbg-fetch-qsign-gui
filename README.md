## 如何启动

- 点击 [Releases](https://github.com/CikeyQi/unidbg-fetch-qsign-gui/releases) 下载最新版本的.zip文件并`解压`，放置在一个 `高性能` 的 `Windows` 服务器上

- 双击打开.exe文件，点击 `添加启动列表`，按提示配置好各项 

- 启动成功后，双击`API地址`列会复制API地址，直接使用即可

- 如需修改配置项，点击相应列即可设置

- 使用期间遇到问题可以加群与我们反馈：**551081559**
